2. Âge minimum

PauseCafé est réservé aux personnes âgées de 9 ans et plus. Aucune donnée n'est collectée sciemment auprès de mineurs. Contacte hello@pausecafe-app.fr pour toute demande de suppression de données d'un mineur.

3. Nature des données collectées

  • Données de compte : prénom, adresse email
  • Données de bien-être : boissons enregistrées, quantités en mg de caféine, horaires
  • Données de progression : points, badges, série, niveau
  • Données de parrainage : code de parrainage personnel (généré automatiquement), code du parrain utilisé à l'inscription le cas échéant, nombre de filleuls parrainés
  • Données de santé indicatives : statut mode précaution (activé/désactivé), date d'activation, objectif mg sauvegardé avant activation — ces données sont strictement personnelles, non partagées, et utilisées uniquement pour adapter les alertes in-app
  • Données techniques : identifiant anonyme Firebase, données de crash anonymisées

Précision : les données de consommation de caféine constituent des données de bien-être personnel. PauseCafé n'est pas un dispositif médical et ne produit pas de conseil médical.

Données mode précaution : le statut du mode précaution (activé/désactivé) peut indirectement révéler un état de santé. Ces données sont traitées avec le même niveau de protection que les données de santé au sens de l'article 9 du RGPD. Base légale : consentement explicite de l'utilisateur (activation volontaire). Ces données ne sont jamais partagées ni vendues.

Données de parrainage : le code de parrainage personnel est visible uniquement par toi dans les réglages de l'application. Le code d'un parrain que tu aurais utilisé à l'inscription est conservé afin de prévenir toute utilisation frauduleuse (un seul code utilisable par compte).

4. Finalités du traitement

  • Fournir le service de suivi de caféine et de gamification
  • Synchroniser les données entre appareils (iPhone, Apple Watch, Widget)
  • Afficher des statistiques personnalisées
  • Envoyer des notifications push (avec ton consentement)
  • Gérer le programme de parrainage et attribuer les récompenses associées
  • Améliorer l'application via Firebase Analytics (données anonymisées)
  • Afficher des publicités via Google AdMob (avec ton consentement ATT)

5. Base légale

  • Exécution du contrat : fourniture du service, gestion du parrainage
  • Consentement : publicités personnalisées et notifications push
  • Intérêt légitime : amélioration via analytics anonymisés, prévention de la fraude au parrainage

6. Sous-traitants

  • Google Firebase — Google Ireland Limited, Dublin, Irlande. Données stockées dans l'UE. Politique Firebase
  • Google AdMob — uniquement avec consentement ATT. Politique Google
  • Apple — Sign in with Apple, notifications. Politique Apple
  • Brevo (ex-Sendinblue) — envoi d'emails transactionnels (confirmation de récompenses). Données : email et prénom uniquement. Politique Brevo

Aucune donnée personnelle n'est vendue à des tiers.

7. Transferts hors UE

Certains sous-traitants peuvent traiter des données hors UE, encadrés par des clauses contractuelles types de la Commission Européenne.

8. Durée de conservation

  • Données de compte : durée d'utilisation active
  • Historique : jusqu'à 2 ans dans l'application
  • Données de parrainage : durée d'utilisation active du compte ; supprimées avec le compte
  • Données mode précaution : supprimées avec le compte ou lors de la désactivation du mode
  • Suppression de compte : données effacées sous 30 jours
  • Données anonymisées : conservation indéfinie

9. Tes droits (RGPD)

  • Accès : export CSV dans l'app (Évolution → Exporter mes données)
  • Rectification : modifier ton profil dans les réglages
  • Effacement : Réglages → Supprimer mon compte
  • Portabilité : export CSV depuis l'app
  • Opposition : Réglages iOS → Confidentialité → Suivi → PauseCafé
  • Retrait du consentement : à tout moment

Contact : hello@pausecafe-app.fr — réponse sous 30 jours.

10. Publicités et tracking (ATT)

  • Refus : publicités non personnalisées, sans tracking
  • Premium : aucune publicité, quel que soit ton choix

Modifier : Réglages iOS → Confidentialité → Suivi → PauseCafé

11. Sécurité

  • Communications chiffrées HTTPS/TLS
  • Données chiffrées au repos via Firebase
  • Mots de passe hachés, jamais stockés en clair
  • Accès aux données de parrainage restreint via les règles de sécurité Firebase

12. Cookies

L'application mobile n'utilise pas de cookies. Le site pausecafe-app.fr est un site statique sans cookie de tracking.

13. Modifications

Cette politique peut être mise à jour. En cas de modification substantielle, une notification sera envoyée via l'application.
