9. Programme de parrainage

Le programme de parrainage est un avantage offert gratuitement à tous les utilisateurs inscrits. Il peut être modifié ou supprimé à tout moment sans préavis.

  • Code unique : un code de parrainage est généré automatiquement à la création de chaque compte.
  • Filleul : toute personne s'inscrivant avec un code valide reçoit +50 points de bienvenue. Un seul code peut être utilisé par compte, uniquement à l'inscription.
  • Parrain : pour chaque filleul inscrit, le parrain reçoit +100 points et 1 mois Premium offert (cumulatif, ajouté à tout Premium existant).
  • Limite : le programme est limité à 10 filleuls maximum par compte. Au-delà, aucune nouvelle récompense n'est accordée.
  • Interdictions : le parrainage de soi-même, la création de comptes fictifs, ou tout usage frauduleux entraîne la suppression immédiate du compte et l'annulation des récompenses.
  • Pas de valeur monétaire : les points et le Premium offert via parrainage n'ont aucune valeur monétaire et ne sont pas remboursables.
  • Modification : l'éditeur se réserve le droit de modifier les conditions, les récompenses ou de mettre fin au programme à tout moment, sans que cela ouvre droit à compensation.
